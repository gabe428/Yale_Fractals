Circle Inversion Fractals

Properties of Inversion

Because dist((a,b), (x,y))*dist((a,b), (x',y')) = r2, we see

    if dist((a,b), (x,y)) > r, then dist((a,b), (x',y')) < r

and

    if dist((a,b), (x,y)) < r, then dist((a,b), (x',y')) > r

That is,

    points outside C invert to points inside C

and

    points inside C invert to points outside C.
