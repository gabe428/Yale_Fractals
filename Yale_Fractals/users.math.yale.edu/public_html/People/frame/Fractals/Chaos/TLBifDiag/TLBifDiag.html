6.J. Bifurcation Diagrams for the

Tent Map and the Logistic Map

Recall the bifurcation diagram is a graph of s-values on the horizontal axis, and the eventual x-values for that s plotted on the vertical line through each s. That is, for each s, iterate x1, ..., x200 say, and plot x100, ..., x200. We are interested in the long-term behavior of the iterates, so usually we drop some initial values.

In dust in the tent we see most points iterate to -infinity for s > 2. Consequently, for the bifurcation diagram we restrict our attention to s in the range 0 <= s <= 2.
