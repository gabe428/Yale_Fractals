Some Algebra of Dimensions

Codimension

The codimension of a set is the dimension of the complementary set. Consequently, the codimension depends on both the dimension of the set and the dimension of the space in which the set is embedded.
If P is a point in the plane, its codimension is 2 - 0 = 2.
If Q is a line in the plane, its codimension is 2 - 1 = 1.
If P is a point in space, its codimension is 3 - 0 = 3.
If Q is a line in space, its codimension is 3 - 1 = 2.
If R is a plane in space, its codimension is 3 - 2 = 1.
