Dimensions and Logarithms

Finding the Dimension
Suppose we have counted the number of scaled copies of an object and we know the scaling factor r. Then we know N(r) and r. Can we find d, the dimension of the object? We have to solve the equation N(r) = (1/r)d for d. This is an exponential equation and we are trying to find the exponent. To solve an equation for one of its variables, we need to do the inverse operations in the reverse order. The inverse of exponentiation is done using logarithms. So we should start to solve for d by taking the log of both sides of the equation. We can use any base for the log we want, as long as it is the same on both sides of the equation.
