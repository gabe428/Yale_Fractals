A Dimension Equation
We have see that for the line segment scaled by 1/n we get
    the number of scaled copies = (1/(1/n))1
For the square we get
    the number of scaled copies = (1/(1/n))2
And for the cube we get
    the number of scaled copies = (1/(1/n))3
It is important to observe that in each case the exponent is the dimension of the object which is scaled. To make the equation neater we use the following variables:
    the dimension of the original object is d
    the scaling factor, 1/n, is r
    the number of scaled copies is N(r)
Then for an object with dimension d which is scaled by r, we have the equation
N(r) = (1/r)d
