Addresses and Functions

The scalings in the centimeter decomposition involve shrinking by 1/10 and placing across the interval. Here are the corresponding functions.
    f0(x) = x/10
    f1(x) = x/10 + 1/10
    f2(x) = x/10 + 2/10
    f3(x) = x/10 + 3/10
    f4(x) = x/10 + 4/10
    f5(x) = x/10 + 5/10
    f6(x) = x/10 + 6/10
    f7(x) = x/10 + 7/10
    f8(x) = x/10 + 8/10
    f9(x) = x/10 + 9/10
So for example, the region with address 37 is f3(f7([0,1])).
