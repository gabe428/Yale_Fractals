9. That the first generator is unifractal means
Log|dY1|/Log(dt1) = Log|dY2|/Log(dt2) = Log|dY3|/Log(dt3) = Log|dY4|/Log(dt4) = H
Then Log|dZi|/Log(dsi) = Log|3 dYi|/Log(dti)
= Log(3)/Log(dti) + Log|dYi|/Log(dti)
= Log(3)/Log(dti) + H
So we see that in order for the second generator to be unifractal, Log(3)/Log(dti) must have the same value for all i, that is, all ti are equal.
