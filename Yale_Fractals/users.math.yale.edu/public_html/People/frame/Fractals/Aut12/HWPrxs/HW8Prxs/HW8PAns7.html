Eighth Homework Set Answers

7. If successive values xi and xi+1 are nearby, then the return map points (xi, xi+1) lie close to the line y = x, so these values generate the left return map.
This was the lie. The center return map works if successive values sum to about 1. If successive values differ by about 1, the return map points (xi, xi+1) lie close to the lines |y - x| = 1, that is, y = x + 1 and y = x + 1.
