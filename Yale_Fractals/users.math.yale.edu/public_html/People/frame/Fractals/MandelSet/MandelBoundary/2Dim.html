The Mandelbrot set and Julia Sets

The boundary of the Mandelbrot set

The boundary of the Mandelbrot set is a very complicated place, because in addition to the Misiurewicz points, every circle centered at every boundary point encloses infinitely many centers, hence infinitely many copies of the Mandelbrot set.
Looking very closely, the branches appear to grow more and more dense. The eye is not mistaken: Shishikura proved the boundary of the Mandelbrot set is 2-dimensional.
