f(alpha) exercise 1 solution

In the formula
alpha(q) = (Log(p1)*p1q + ... + Log(p1)pNq)/(log(r)*(p1q + ... + pNq))
note 0 < r < 1 and each 0 < pi < 1 so Log(r) < 0 , each Log(pi) < 0, and each piq > 0. Consequently, the denominator is negative and each term of the numerator is negative.
That is, for each q, alpha(q) is positive.
