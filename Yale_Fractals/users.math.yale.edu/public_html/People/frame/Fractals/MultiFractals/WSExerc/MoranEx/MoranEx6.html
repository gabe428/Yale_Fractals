Eight pieces, r1 = ... = r4 = 1/4, r5 = ... = r8 = 1/8. The Moran equation is
4 (1/4)d + 4 (1/8)d = 1
Taking x = (1/2)d, this becomes
4 x3 + 4 x2 - 1 = 0.
The real solution is
x = (4 - 2 y + y2)/(6 y), where y = (19 + 3 sqrt[33])1/3
so x is approximately 0.419643
and so the dimension is
d = Log(0.419643)/Log(1/2), approximately 1.25277.
