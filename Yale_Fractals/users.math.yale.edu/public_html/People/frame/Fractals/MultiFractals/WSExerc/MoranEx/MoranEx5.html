Twelve pieces, r1 = ... = r4 = 1/4, r5 = ... = r12 = 1/16. The Moran equation is
4 (1/4)d + 8 (1/16)d = 1
Taking x = (1/4)d, this becomes
8 x2 + 4 x - 1 = 0.
The positive solution is
x = (-4 + sqrt(48))/16 = (-1 + Sqrt[3])/4
and so the dimension is
d = Log((-1 + sqrt(3))/4)/Log(1/3), approximately 1.22499
