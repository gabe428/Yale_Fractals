First, recall dtau/dq < 0 and d2tau/dq2 >= 0, so dtau/dq is decreasing.
Next, from
dhi/dq = piqritau(q) (ln(pi) + ln(ri)(dtau/dq))
we see dhi/dq = 0 exactly when ln(pi) + ln(ri)(dtau/dq) = 0
Because both ln(pi) < 0 and ln(ri) < 0, dhi/dq = 0 requires dtau/dq < 0, which has already been established.
Because dtau/dq is decreasing, we see ln(pi) + ln(ri)(dtau/dq) = 0 for at most one q.
Consequently, dhi/dq has at most one zero.
