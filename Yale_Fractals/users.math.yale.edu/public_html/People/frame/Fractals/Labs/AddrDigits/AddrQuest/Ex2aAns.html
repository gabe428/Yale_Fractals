Some Basic Questions on Decimals through Addresses

2. (a) The left endpoint of the interval with address 0110 is
0*(1/2) + 1*(1/4) + 1*(1/8) + 0*(1/16) = 3/8.
The interval with address 0 has length 1/2, the interval with address 01 has length 1/4, and so the interval with address 0110 has length 1/16. Consequently, if x lies in the interval with address 0110, then
3/8 <= x <= 3/8 + 1/16 = 7/16.
