Some Basic Questions on Decimals through Addresses

1. (d) Reading from left to right, each successive address digit places a point in a smaller interval; that is, locates the point more precisely.
Postal addresses are written in the opposite order.
For example, an initial segment of 3312 Grant Avenue, St. Albans, WV, USA would locate 3312 Grant Avenue, but which Grant Avenue?
Probably that is a street name of a street in hundreds of cities.
This information does not restrict a search to any geographical area.
Knowing 3312 Grant Avenue, St. Albans is more precise, but does it refer to St. Albans in WV, or VT, or in England?
By contrast, if the address of a point is 10, we know it lies in the interval [1/2, 3/4].
