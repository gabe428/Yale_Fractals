Some Basic Questions on Decimals through Addresses

1. (b) From left to right, the addresses of the intervals are 00, 01, 10, and 11.
These digits can be viewed as a binary representation of the number of 1/4s of the points in each interval.
For example, for 1/2 < x < 3/4 we see x = 2*(1/4) + y, where 0 < y < 1/4.
