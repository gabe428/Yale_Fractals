Gaskets by Forbidden Triples

These forbidden triples produce a standard gasket. Let's see how.
To produce this gasket with forbidden pairs, we need 1 → 4, 2 → 4, 3 → 4, and 4 → 4 forbidden. That is, T4 cannot follow T1, T2, T3, or T4.
From the software description, we see that to forbid all triples i → j → 4, we set to 0 all entries in the fourth face.
Setting to 0 all the entries of any other three faces produces each of the other three gasket orientations.
