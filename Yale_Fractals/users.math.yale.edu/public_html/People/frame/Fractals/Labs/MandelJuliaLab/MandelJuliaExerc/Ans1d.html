Julia Sets and the Mandelbrot Set

Exercise Answers

1. (d) Say N is the cycle number of the disc to which the antenna is attached.
Except for the spoke attaching the antenna to the body of the Mandelbrot set, the largest midget in one spoke has cycle number N + 1, the largest in another has cycle number N + 2, ..., the largest midget in the remaining spoke has cycle number 2*N - 1.
(The order in which these cycles are arranged can be determined, but requires a bit more work.)
Proceeding along each spoke to the point where the spokes join, the cycle of the largest midget increases by N.
For the spoke attaching the antenna to the body of the Mandelbrot set, the largest midget has cycle number N*3.
